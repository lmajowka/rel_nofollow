# RelNofollow

Add rel="nofollow" to every link in a string


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rel_nofollow'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rel_nofollow

## Usage

simply call

          string.add_nofollow
